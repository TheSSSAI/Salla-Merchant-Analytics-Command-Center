Responsive Specifications

Mobile Adaptations

  • Toolbar items wrap to multiple rows or scroll horizontally
  • Padding reduced to maximize content area
  • Action buttons stack vertically in footer

Tablet Adaptations

  • Toolbar remains sticky on scroll
  • Standard padding restored

Desktop Enhancements

  • Full formatting toolbar visible
  • Side-by-side preview mode option
  • Keyboard shortcuts enabled (visual cues)

Accessibility Features

Toolbar navigation via arrow keys, standard rich text shortcuts (Ctrl+B/I)

Screen Reader Support

Toolbar labeled via aria-label, toggle buttons indicate aria-pressed state

Focus Management

Visible focus rings on all interactive elements

Aria Attributes

Usage Guidelines

When To Use

For creating or editing email marketing templates, cart recovery sequences, or system notifications.

Content Guidelines

Provide clear labels for 'Template Name' and 'Subject'. Editor area should have a min-height.

Placement Rules

Typically the main content of a configuration page (full width).

Customization Options

  • Toggle Preview Mode
  • Save/Cancel Actions
